Mastering Feedback - Implementing Thumbs Up/ Thumbs Down

  • What is it?
    • The 'Thumbs Up/Thumbs Down' icons are available after each response given in the 'Seek' tab of NeuralSeeks UI. These responses indicate a score of 5 for Thumbs Up and 0 for Thumbs Down. Theses icons are able to be added to a virtual agent of choice to be shown and utilized in-line with the conversation.
  • Why is it important?
    • The Thumbs Up/Thumbs Down icons within NeuralSeek are userful for clients to be able to provide feedback to answers generated by NerualSeek based on queries relevant to their connect corporate content. Being able to implement these icons into a virtual agent is important for clients who want to provide their users with a way to provide relevant, trackable feedback.
  • How does it work?
    • After a query is submitted in NeuralSeek's 'Seek' tab, users can simply click either the 'Thumbs Up' icon or the 'Thumbs Down' icon based on their impression of the generated response. The response is then tracked and recorded within the relevant intent on NeuralSeek's 'Curate' tab. Users are able to implement the icons into a virtual agent by using the uniquely generated SVG URL provided after each response. See below for information on using 'iframe' response type to integrate these feedback icons within the IBM virtual agent watsonx Assistant.

Adding an iframe response in watsonx Assistant

  • What is it?
    • 'iframe' embeds content from an external website, such as a form or other interactive component, directly within the chat. The source content must be publicly accessible using HTTP, and must be embeddable as an HTML iframe element.
  • Why is it important?
    • An iframe response is useful if you want to enable customers to perform some interaction with an external service without leaving the chat. In this example, users can use an iframe response to display the thumbs up/thumbs down icons within the web chat.
  • How does it work?
    • This reponse type embeds content from another website directly inside the chat window as an HTML 'iframe element'. It can be displayed in the chat window as a clickable preview card that describes the content, or inline within the conversation.
  1. Navigate to the watsonx Assistant Instance, and open an Action.
  2. In the 'Assistant says' field within the relevant conversation step, click the 'iframe' icon.
  3. Add the full URL to the external content in the 'iframe source' field.
    • SVG files containing appropriate URL can be requested from general@cerebralblue.com.
  4. Users can include a query parameter for background-color to the thumbs url given: ' ?style=background-color%3A%23f4f4f4 '
  5. Put it as inline with a height of 45
  6. Optionally, add a descriptive title in the 'Title' field.
